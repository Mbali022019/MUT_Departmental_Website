## Project Overview

The MUT ICT Departmental Website is a modern and responsive web application developed to improve communication, accessibility, and user interaction within the Mangosuthu University of Technology ICT Department.

The website provides students, staff members, and visitors with easy access to departmental information, academic programmes, announcements, and login portals.

This project was developed as a group project by five members, where each member contributed to different sections of the system including the homepage design, student login, staff login, responsiveness, and overall website functionality.

---

# Features

* Modern and responsive homepage
* Professional navigation bar
* Student login system
* Staff login system
* Interactive user interface
* Responsive design for desktop and mobile devices
* Organized programme and department sections
* Smooth navigation and user-friendly layout
* Search functionality

---

# Technologies Used

The following technologies were used to develop the project:

* HTML5
* CSS3
* JavaScript
* Bootstrap / Tailwind CSS
* Responsive Web Design Principles

---

# System Objectives

The main objectives of the project are:

* To provide a digital platform for the ICT Department
* To improve communication between students and staff
* To provide easy access to departmental information
* To create a modern and professional departmental website
* To improve user experience through responsive design
